# 🚀 Personal Developer Portfolio

A modern developer portfolio built with Next.js, focused on showcasing **client work, open source contributions, and personal projects** in a clean, structured layout.

---

## ✨ Features

* 🧩 **Segmented project showcase**

  * Client Work
  * Open Source Contributions
  * Personal Projects
* 📂 Expandable project sections (clean UI, no clutter)
* 🎨 Minimal dark UI inspired by GitHub
* ⚡ Fast and optimized with Next.js App Router
* 📊 Integrated analytics (Vercel Analytics)
* 🔗 External links to live projects and repositories

---

## 🛠 Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Language:** TypeScript
* **Deployment:** Vercel

---

## 📸 Overview

This portfolio is designed to clearly communicate:

* 💼 Real-world client work
* 🌍 Open source contributions
* 🧠 Independent projects

Instead of listing everything together, projects are **intentionally grouped** to highlight different strengths.

---

## 🚀 Getting Started

Clone the repo:

```bash
git clone https://github.com/abusaeed-shuvo/abusaeed-portfolio.git
cd abusaeed-portfolio
```

Install dependencies:

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

---

## 📦 Build

```bash
npm run build
npm start
```
